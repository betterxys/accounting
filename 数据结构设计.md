## 🔄 业务流程

### 1. 月度记账流程
1. 选择记账日期（通常是月末）
2. 依次录入肖肖和运运各账户余额
3. 系统自动计算：
   - 个人总资产
   - 家庭总资产  
   - 相比上月变化
4. 保存月度记录

### 2. 数据分析流程
1. **月度对比**：当月 vs 上月的变化
2. **趋势分析**：最近6-12个月的资产走势
3. **账户分布**：各类账户占比分析
4. **个人对比**：肖肖 vs 运运的资产情况

## 📈 图表设计

### 1. 资产趋势图
- X轴：月份
- Y轴：资产金额
- 线条：肖肖、运运、总计（三条线）

### 2. 账户分布饼图
- 按账户类型显示占比
- 支持肖肖/运运分别查看

### 3. 月度变化柱状图
- 显示每月的增减情况
- 正值（绿色）表示增长，负值（红色）表示减少

## 🔧 技术实现

### 前端
- HTML5 + CSS3 + JavaScript
- Chart.js 用于图表绘制
- 响应式设计，支持手机访问

### 后端存储
- 微信云开发版本：云数据库
- 本地调试版本：localStorage
- Node.js版本：JSON文件

### 数据库表结构
1. `monthly_records` - 月度记录
2. `account_types` - 账户类型配置  
3. `system_settings` - 系统设置

这个设计完全符合你们的使用场景：定期记录、双人管理、趋势分析！
