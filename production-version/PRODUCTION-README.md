## 使用说明
1. **首次使用**: 直接打开 index.html 或运行启动脚本
2. **日常记账**: 建议每月记录一次各账户余额
3. **数据备份**: 定期使用设置页面导出数据
4. **多设备同步**: 导出数据文件，在其他设备导入

## 部署建议
### 本地使用
- 双击 `index.html` 直接使用
- 或运行 `./start.sh` 启动本地服务器

### 服务器部署
- 将所有文件上传到网站根目录
- 确保服务器支持静态文件访问
- 可配置 HTTPS 确保数据安全

### 微信小程序
- 可移植到微信小程序平台
- 需要配置微信云开发数据库
- 数据结构已兼容云数据库格式
