# 💰 肖肖&运运专用资产管理系统

专为肖肖和运运设计的双人资产管理系统，支持月度记账、趋势分析和可视化图表。

## 🎯 核心功能

### 📊 资产概览
- 显示肖肖和运运各自的总资产
- 显示家庭总资产
- 自动计算相比上月的增减变化
- 资产趋势图预览

### 📝 月度记账
- 支持肖肖和运运分别记录各账户余额
- 预设常用账户：招商银行、中国银行、建设银行、微信、支付宝、现金
- 支持自定义添加账户类型
- 实时计算个人和家庭总资产
- 自动保存每月记录

### 📈 数据分析
- **资产趋势图**：查看最近6-24个月的资产变化曲线
- **账户分布图**：饼图显示各账户类型的资产占比
- **月度变化图**：柱状图显示每月的资产增减情况
- **个人对比图**：对比肖肖和运运的资产情况

### ⚙️ 系统设置
- 账户类型管理（添加/删除自定义账户）
- 数据导出/导入（JSON格式）
- 数据备份和恢复
- 系统信息查看

## 📱 使用指南

### 首次使用
1. 打开系统，会看到空白的概览页面
2. 点击右上角"📝 记账"按钮
3. 选择记账日期（建议每月最后一天）
4. 分别输入肖肖和运运各账户的余额
5. 点击"💾 保存记录"

### 日常使用
1. **月度记账**：每月记录一次各账户余额
2. **编辑记录**：可以修改已录入的历史记录
3. **查看趋势**：在"📈 分析"页面查看资产变化趋势
4. **账户管理**：在"⚙️ 设置"页面添加新的银行账户

### 编辑已有记录
1. **在概览页面**：点击最近记录右侧的"✏️"按钮
2. **在记账页面**：选择之前的日期，系统自动加载已有数据
3. **编辑完成**：点击"💾 更新记录"保存修改
4. **删除记录**：在概览页面点击"🗑️"按钮（谨慎操作）

### 数据管理
- **导出数据**：定期导出数据文件作为备份
- **导入数据**：可以导入之前的备份文件
- **清空数据**：慎用，会删除所有记录

## 💡 使用建议

### 记账频率
- **推荐**：每月月底记录一次
- **最低**：每季度记录一次
- **最高**：每周记录一次

### 账户设置
- 根据实际使用的银行卡和支付工具添加账户
- 可以删除不使用的预设账户
- 建议为每个账户设置不同的图标和颜色便于区分

### 数据安全
- 定期导出数据备份到云盘
- 数据存储在浏览器本地，清除浏览器数据会丢失记录
- 重要：换电脑前务必导出数据

## 🎨 界面说明

### 概览页面
- **资产卡片**：显示个人和家庭资产总额
- **增减指示**：绿色表示增长，红色表示减少
- **趋势图**：最近6个月的资产变化曲线
- **最近记录**：显示最近5次记账记录

### 记账页面
- **日期选择**：选择本次记账的日期
- **双人区域**：左侧肖肖，右侧运运
- **实时计算**：输入金额时自动计算总计
- **保存记录**：检查无误后保存本月记录

### 分析页面
- **时间范围**：可选择6/12/24个月的数据范围
- **四种图表**：趋势、分布、变化、对比
- **交互式图表**：可以悬停查看具体数值

### 设置页面
- **账户管理**：查看、添加、删除账户类型
- **数据操作**：导出、导入、清空数据
- **系统信息**：查看数据统计和最后记账时间
